## How to use pyqt5-tools?

1. open ```designer``` with in the venv via terminal
2. draw the UI and save it to ```./ui/example.ui```
3. use CLI ```pyuic5 -x ./ui/example.ui -o ./pyui/example.py``` to generate python version of ui
